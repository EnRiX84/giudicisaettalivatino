## 9. Punti Deboli del Design Attuale

- Template Joomla datato (estetica anni 2010)
- Layout sovraccarico senza gerarchia visiva
- Nessun design responsive moderno (mobile-first)
- Colori piatti (rosso #DD514A, bianco, blu link), nessuna animazione
- Notizie in elenco verticale senza card o griglie
- Footer disorganizzato con muro di banner
- Mancanza di hero section e call-to-action chiare
- Navigazione poco intuitiva su mobile
- Nessuna sezione visiva per l'offerta formativa
- PDF e documenti difficili da trovare

---

## 10. Piano di Redesign Proposto

### Struttura Nuova
1. **Hero Section** - Immagine scuola, nome, motto, CTA iscrizioni
2. **Navbar moderna** - Sticky, responsive, hamburger menu, dropdown
3. **Sezione Notizie** - Card grid (immagine, titolo, data, anteprima)
4. **Offerta Formativa** - Card dedicate per ogni indirizzo con icone
5. **Sezione Documenti** - Organizzata per categorie con filtri
6. **Sezione Progetti** - Erasmus+, PNRR, PON, Biologia Biomedica
7. **Footer strutturato** - Colonne (contatti, link rapidi, loghi, servizi)
8. **Barra servizi rapidi** - Registro, Gmail, Modulistica, WhatsApp

### Tecnologie Previste
- HTML5 + CSS3 + JavaScript vanilla (sito statico)
- Design responsive mobile-first
- Animazioni CSS sottili
- Font moderni (Google Fonts)
- Palette colori rinnovata ma coerente

### Contenuti da Preservare
- TUTTE le notizie e circolari con link ai PDF originali
- Menu di navigazione completo
- Link servizi esterni (Registro, Gmail, Albo Pretorio, ecc.)
- Documenti amministrativi, graduatorie, contratti
- Sezioni Erasmus+, PNRR, PON, PCTO
- Contatti completi (indirizzi, telefoni, email)
- Banner istituzionali obbligatori
- Risorse antibullismo
- Area BES/DSA, Biblioteca, Sicurezza
